Использование Postman
Postman — удобный графический интерфейс для тестирования API. Он позволяет легко отправлять запросы, управлять параметрами и просматривать ответы.

Шаги для отправки запроса в Postman:
Установите Postman :
Скачайте и установите Postman с официального сайта .
Создайте новый запрос :
Откройте Postman и создайте новый запрос, нажав на кнопку "New" и выбрав "Request".
Настройте запрос :
Введите URL вашего API, например, http://localhost:8080/get-resume.
Выберите метод POST.
Добавьте тело запроса :
Перейдите на вкладку "Body".
Выберите "form-data".
Добавьте два ключа:
Ключ : data
Тип : File
Значение : выберите ваш JSON файл (например, data.json).
Ключ : file
Тип : File
Значение : выберите ваш файл резюме (например, resume.pdf).
Отправьте запрос :
Нажмите кнопку "Send" для отправки запроса.
Просмотрите ответ от сервера.
